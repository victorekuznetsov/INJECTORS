## Технические детали для продолжения

- Тестовый цикл: скопировать `dashboard_injectors.html` → `test_dashboard.html`, заменить CDN
  Chart.js на локальную копию `scratchpad/package/dist/chart.umd.js` (CDN недоступен в песочнице),
  прогнать через Playwright (`executable_path='/opt/pw-browsers/chromium'`), удалить
  `test_dashboard.html` перед коммитом.
- Порядок публикации: правка `dashboard_injectors.html` → пересобрать `dashboard_injectors.zip`
  (`zip -q dashboard_injectors.zip dashboard_injectors.html`) → commit+push на
  `claude/dashboard-injectors-setup-vu2r74` → зеркалировать в `Report/dashboard_injectors.zip` на
  `main` (через временный `git worktree`, отдельная ветка → push прямо в `main`) → отправить zip
  пользователю.
- Схема `VALVES[m].cyl[cylId]`: `inClr`/`exClr` (мм), `h:[В,Н]` (мм, абсолютная высота),
  `exH`/`inH` (мм, для формата `sag`), `exSettled`/`inSettled` (клапан закатан), `inIneq`/`exIneq`
  (`'>'`/`'<'`, когда источник дал только неравенство).
- `cylNum(id)`: интерливинг 1L=1,1R=2,2L=3,2R=4,...,8L=15,8R=16 — подтверждён независимо и
  КАМСС-отчётом ("1L (1) 5L (9) 1R (2) 5R (10)...").
